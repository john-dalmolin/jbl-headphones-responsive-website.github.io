# JBL Headphones Responsive Website

Landing page responsiva inspirada na identidade visual da JBL, desenvolvida com foco em UI moderna, organização de código e experiência mobile-first.

## Demo

Acesse o projeto publicado:
[Live Demo](https://john-dalmolin.github.io/jbl-headphones-responsive-website.github.io/)

## Visão Geral

Este projeto foi construído para praticar fundamentos de front-end em um cenário real:
- Estrutura semântica com HTML5
- Estilização responsiva com CSS3
- Interações e animações com JavaScript

## Funcionalidades

- Layout responsivo para desktop, tablet e mobile
- Componentes visuais com foco em apresentação de produto
- Animações para melhorar a experiência do usuário
- Estrutura simples e fácil de evoluir

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla JS)
